# Atena Game Engine

## Building

### Dependencies

* [Buck](buckbuild.com)
* [SDL2](https://www.libsdl.org/)
* [SDL2_image](https://www.libsdl.org/projects/SDL_image/)
* [Box2D](http://box2d.org/)
* [Jansson](http://www.digip.org/jansson/)

### Compile and run

```sh
buck
./buck-out/gen/atena
```
